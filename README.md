# Fit Zone Backend Microservices

This repository contains all the microservices for the backend of the **Fit Zone** application.

## Overview

Each service is designed to handle a specific part of the system, following a microservices architecture. The goal is to ensure scalability, maintainability, and separation of concerns across the backend.

## Features

- Modular microservice structure
- RESTful APIs
- Easy to scale and deploy independently
- Written in FastAPI
